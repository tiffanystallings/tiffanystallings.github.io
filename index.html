Movie Trailer Website

Python, HTML, CSS

Udacity's Requirements:

I was asked to use object-oriented Python to create objects that held movie data in server-side files. I was then to pass a list of these objects to fresh_tomatoes.py -- a bit of Python code provided by Udacity to turn the movie data into an html file that shows movie poster images and plays trailers.

Extra Credit:

The posters for my movie were all relatively dark, so I darkened the color scheme considerably. The header was a Bootstrap navbar with preset colors, so I decided to remove its links to Bootstrap and make it a simple header instead. I also imported a fun Google Font to switch things up.

More interestingly, I was able to further alter fresh_tomatoes.py to collect storyline information from the movie objects and used that to add the ability to see a movie's storyline on hover.

Check out the live demo!

Classic Arcade Game Clone

Javascript, HTML Canvas

Udacity's Requirements:

I needed to use the provided engine and graphics to create a game where the player must avoid moving bugs and get to the water. Upon collision with the bugs, the game would reset. Upon getting to the water, the game would display "You Win".

Extra Credit:

This was a lot of fun to build, but as an avid gamer I knew I wanted to strive for more. I created a water oject that -- like the bugs -- causes loss of life. Instead of winning by getting to the water, I created a star object that must be obtained to advance to the next level. I changed how the engine lays out levels in order to make the creation of additional levels much more simple.

To add an additional challenge, I created gem objects than can be obtained to score additional points. And instead of starting the game over entirely on colliding with a bug or the water, I added lives that were tracked over all levels. Lose three lives, and you lose the game.

Check out the live demo!

Interactive Resume

Javascript, HTML, CSS

Udacity's Requirements:

I was provided with boilerplate html and CSS and was tasked with filling in a Javascript file which would target the DOM elements and iteratively fill them in with my resume Information. I also needed to work out how to implement the Google Maps API at the bottom of the page.

Extra Credit:

The original template's style was a tad on the masculine side, so I added a lot of my own colors to it.

I also added a small jQuery feature that changes the project images on click. When I revisit this project to add updated information, I intend to use this feature to highlight different elements in my projects that I particularly enjoy.

One last little challenge from Udacity was to read through the documentation and get the pins of the Google Maps API to show additional information. Mine simply tell the user the precise city the pin is attached to.

Check out the live demo!

Restaurant Menu Database

Python, Flask, SQLAlchemy

Udacity's Requirements:

I was to create an item database which allowed users to log in and out securely via OAuth2 providers. Once signed in, users could add, edit, and delete items. Items created by users are protected against any editing or deleting by other users.

Extra Credit:

I went a little further by modularizing this application. I created a base template with the header and login functionality to extend all other templates, and moved authorization and CRUD functions into their own modules. The templates respond to user logins by welcoming the user, and only providing links to pages that they are authorized to access.

Check out the live demo!

Data Analysis

Python, PostgreSQL, Psycopg2

Udacity's Requirements:

Provided with a PostgreSQL database from a fake news website, I was to write queries that would get information on articles with the most clicks, authors with the most article clicks, and days on which a high number of errors occured. I then ran those queries in python code using Psycopg2 to provide results in the command line.

Move Planner

HTML, JavaScript, Google Streetview API, New York Times API, Wikipedia API

Built alongside the Udacity course on building AJAX requests, this simple one-page application aims to provide the user with information about a location they plan on moving to. Upon entering a location and clicking Search, the banner will update with a Google Streetview image of that location, and the user will be provided with search results of that location from New York Times and Wikipedia.

Lightweight Geocoder

HTML, JavaScript, Wordpress, Google Maps Geocode API

Previously, the employees of Broadband Network Solutions would manually search locations on Google Maps in order to acquire their coordinates. Their IT Manager saw an opportunity to streamline by building a geocoder into their office homepage on Wordpress, and came to me for a solution.

I created for them a simple JavaScript plugin build from the Google Maps Geocode API that would allow them to quickly enter in locations and copy those locations from the list of results. This dramatically cuts down on the time it previously took them to acquire coordinates.

Check it out on the BNS website!
